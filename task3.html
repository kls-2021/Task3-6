"JAVA SCRIPT(not-defined,undefined,defined)"

not defined:

If the variable name which is being accessed doesn't exist in memory space then it would be "not defined".

undefined:

If the variable name which is being accessed exists in memory space but hasn't been assigned any value till now, then it would be "undefined".

defined:

If the variable name which is being accessed exists in memory space and has been assigned a value, then it would be "defined".
